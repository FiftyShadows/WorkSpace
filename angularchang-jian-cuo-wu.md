##Angular使用less管理css样式

首先需要修改angular-cli.json中的配置信息

```
 "apps": [
    {
      "styles": [
        "styles.less"
      ]  
  ],
  "defaults": {
    "styleExt": "less",
  }
```

由于改变了Angular的配置中的css为less,因此 你所有的css后缀文件都要改成less。
