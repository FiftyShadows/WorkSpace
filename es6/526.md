##Iterator的基本用法

- Iterator在数组中的应用

```
{
    let arr = ['hello', 'world'];
    let map = arr[Symbol.iterator]();
    console.log(map.next());    //Object {value: 'hello', done: false}
    console.log(map.next());    //Object {value: 'world', done: false}
    console.log(map.next());    //Object {balie: undefined, done: true}
}
```
